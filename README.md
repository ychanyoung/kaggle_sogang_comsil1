2025년 3월 24일 오후 3시에 시작되어, 3월 28일 오후 6시에 종료됩니다. 2주간 실습했던 내용들을 바탕으로 머신러닝 예측 모델로 실생활에 적용되는 문제를 풀어보는 것을 목적으로 합니다.

이 대회에는 총 48,000개 정도의 수집된 로그 데이터가 주어집니다. 해당 로그 데이터에는 어떤 유저가 특정 아이템을 조회한 기록이 주어집니다. 여러분이 Amazon, Coupang 같은 e-commerce 사이트를 사용한다고 가정해봅시다. 온라인 쇼핑 플랫폼을 운영하는 사이트에 데이터베이스는 유저들이 웹페이지에서 활동한 모든 기록이 log로 저장됩니다. 이 log를 살펴보면 사용자들의 구매 행동을 분석할 수 있습니다.

우리는 e-commerce 회사의 Data Scientist로 어떤 유저가 어떤 아이템을 클릭할지 예측하고 싶습니다. 어떤 아이템을 클릭할지 알 수 있다면 더 개인화된 추천을 할 수 있습니다.

다만, 문제가 한 가지 있습니다. 모든 유저가 로그인을 한 상태에서 플랫폼을 사용한다면 좋겠지만 그렇지 않은 경우도 많습니다. 그래서 우리는 로그인 정보가 없는, 즉 유저 정보가 없는 로그 데이터만을 가지고 클릭 여부를 예측하는 문제를 풀고자합니다.
